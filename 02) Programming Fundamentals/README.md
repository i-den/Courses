 ## *The second course introducing different data structures and objects*
>  ### *Primitive Data Types*
 ```C#
 int a = 1;
 double b = 1.5;
 char c = 'a';
 string d = "Name";
 bool e = true;
 
 ```
> ### *Simple math operators*
 ```C#
 var a = 3;
 var b = 4;
 var c = a + b;
 
 Console.WriteLine(c);
 ```
> ### *Simple Conditional Statements*
 ```C#
 if(false)
 {
 }
 else if(false)
 {
 }
 else(true)
 {
 }
 
 switch()
 {
 case true:
 break;
 default:
 break;
 }
 ```
> ### *Simple Loops*
 ```C#
 for (int i = 0; i < num; i++) 
 {
 }
 
 while (true)
 {
 }
 ```
