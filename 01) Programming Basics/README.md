> ```C#
> Console.WriteLine();
> Console.ReadLine();
> ```
> 2. Simple math operators
> ```C#
> var a = 3;
> var b = 4;
> var c = a + b;
> Console.WriteLine(c);
> ```
> 3. Simple Conditional Statements
> ```C#
> if(false)
> {
> }
> else if(false)
> {
> }
> else(true)
> {
> }
> switch()
> {
> case :
> break;
> }
> ```
> 4. Simple Loops
> ```C#
> for (int i = 0; i < num; i++) 
> {
> }
> while (true)
> {
> }
> ```
