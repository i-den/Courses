 ## *The introductory course to programming dealing mostly with:*
>  ### *Console, using C# and Windows*
 ```C#
 Console.WriteLine();
 Console.ReadLine();
 ```
> ### *Simple math operators*
 ```C#
 var a = 3;
 var b = 4;
 var c = a + b;
 
 Console.WriteLine(c);
 ```
> ### *Simple Conditional Statements*
 ```C#
 if(false)
 {
 }
 else if(false)
 {
 }
 else(true)
 {
 }
 
 switch()
 {
 case true:
 break;
 default:
 break;
 }
 ```
> ### *Simple Loops*
 ```C#
 for (int i = 0; i < num; i++) 
 {
 }
 
 while (true)
 {
 }
 ```
